Application DESKTOP JAVA gestion de restauration d'un hopital

Le but du projet était de proposer une application Desktop en JAVA qui permettait la gestion des repas des patients d'un hôpital.

Le personnel hospitalier de chaque service devait pouvoir selectionner un type de repas pour chaque patient de son service.

Le personnel de restauration devait pouvoir visualiser les repas de tous les patients selon le service et pouvoir obtenir un récapitulatif de ceux-ci en PDF.
